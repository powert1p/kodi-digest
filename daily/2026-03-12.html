🔥 НОВОЕ
/batch + /simplify — параллельные агенты из коробки
В Claude Code появились две команды-бомбы. /batch — описываешь N задач, Claude спавнит отдельного агента на каждую в изолированном git worktree, каждый создаёт отдельный PR. /simplify — запускает 3-агентный ревью-пайплайн на твои изменения перед коммитом. /batch автоматически прогоняет /simplify на каждом PR.
Что ещё: HTTP Hooks (POST JSON на URL вместо shell-скриптов), конфиги теперь шарятся между worktrees одного репо, нативное управление MCP через /mcp в VS Code.

AdCP — открытый протокол для рекламных агентов
Ad Context Protocol (AdCP) — открытый стандарт на базе MCP специально для рекламных workflow. Стандартизирует обмен данными об аудиториях, инвентаре и целях кампаний между AI-агентами. 4 модуля: Signals (аудитории), Media Buy (исполнение), Creative (ассеты), Curation (Q2 2026).
Кто за этим: AgenticAdvertising.org, поддержка паблишеров, платформ, агентств. Код на GitHub. По сути MCP, но с рекламной семантикой — стандарт для всей индустрии.

🔥 ПРИОРИТЕТ
/batch + /simplify → ускорение разработки CDP в 3-5x
Вместо последовательной работы агентов — даёшь 5 задач, /batch спавнит 5 параллельных агентов в изолированных worktrees. Каждый прогоняет 3-агентный ревью (/simplify) и создаёт PR. Ты просто ревьюишь и мержишь готовые PR.
Как внедрить
1. Обновить Claude Code до последней версии: npm update -g @anthropic-ai/claude-code
2. В CDP проекте: /batch "Добавь endpoint /api/segments" "Создай React компонент SegmentList" "Напиши миграцию для таблицы segments" "Добавь тесты для segments service"
3. Каждая задача = отдельный PR с автоматическим ревью
⏱ 5 минут 💰 $0 (уже есть подписка) 📈 3-5x ускорение
