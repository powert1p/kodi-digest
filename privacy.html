Условия использования
Какие данные собираем
Kodi Digest собирает минимальный набор данных для работы:
  • → Лайки и дизлайки на карточки дайджеста
  • → Задачи, добавленные в бэклог
Зачем
Данные используются исключительно для:
  • → Персонализации дайджеста под ваши интересы
  • → Трекинга задач в вашем бэклоге
Где храним
Все данные хранятся в двух местах:
  • → Notion workspace владельца интеграции
  • → localStorage вашего браузера (локально на вашем устройстве)
Третьи стороны
Мы не передаём ваши данные третьим лицам. Никакой аналитики, рекламных трекеров или внешних сервисов.
Контакт
Вопросы и обращения — через GitHub Issues в репозитории проекта.
